01
一页一主角
One Sovereign Per View
每页只有一个视觉锚点。图表页主角是图表，侧栏、标题、工具栏都是配角。主角出场时，配角退后。
Every view has exactly one visual anchor. When the chart speaks, the sidebar listens. The supporting role exists only to elevate the protagonist.
02
细节即信仰
Details Are Dogma
1px 的偏差、0.5 度的灰阶差、2px 的对不齐——这些不是 bug，是信仰的裂缝。完美不是没有东西可加，而是没有东西可减。
A 1px misalignment is not a bug, it's a crack in faith. Perfection is achieved not when there's nothing left to add, but when there's nothing left to take away.
03
留白即呼吸
Space Is Breath
密集是焦虑的，松弛是自信的。间距不是浪费，是沉默的音符。4px 基数体系确保每一次呼吸都节奏一致。
Density is anxiety. Space is confidence. Our 4px baseline ensures every breath follows the same rhythm — generous, deliberate, and consistent.
04
克制即力量
Restraint Is Power
全界面只有红绿两种有彩色。两个颜色让你做更好的决策——90% 的中性色 + 10% 的强调色 = 100% 的清晰度。
Only two accent colors across the entire surface. Two colors force better decisions. 90% neutral + 10% accent = 100% clarity. Constraints are not limitations; they are liberation.
风格锚定：Linear 的暗色克制 + Stripe 的排版精度 + 日本设计 的减法美学 Anchored by Linear's dark restraint, Stripe's typographic precision, and Japanese "subtraction" aesthetics.

快速开始
Getting Started

三行命令，拥抱极致 Three commands to embrace the extreme

# 安装设计 Token# Install design tokens
npm install @libra/tokens

# 在入口文件引入# Import in your entry
import '@libra/tokens/css'
# 主题引擎（浏览器环境）# Theme engine (browser)
npm install @libra/theme

import { applyPreset } from '@libra/theme'
applyPreset('dark'); // 🌙 暗色// 🌙 Dark
applyPreset('soft'); // 🌤 柔光（默认）// 🌤 Soft (default)
applyPreset('light'); // ☀ 亮色// ☀ Light
# CLI 安装组件（类似 shadcn）# CLI components (shadcn-style)
npx @libra/cli init
npx @libra/cli add button
npx @libra/cli add card
npx @libra/cli add table
AI 集成
AI Integration

给 AI 装上 Libra 的眼睛 Give AI eyes for Libra

@libra/mcp-server 将设计系统暴露为 MCP（Model Context Protocol）工具，让 Claude Code、Cursor 等 AI 编码助手直接生成符合 Libra 设计语言的 UI 代码。

@libra/mcp-server exposes Libra's design tokens as MCP (Model Context Protocol) tools, enabling AI coding assistants like Claude Code and Cursor to generate UI code that follows Libra's design language.

提供的工具 · Available Tools
