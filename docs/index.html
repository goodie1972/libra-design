设计哲学
Philosophy

01
一页一主角
One Sovereign Per View
每页只有一个视觉锚点。图表页主角是图表，侧栏、标题、工具栏都是配角。
Every view has exactly one visual anchor. When the chart speaks, the sidebar listens.
02
细节即信仰
Details Are Dogma
1px 的偏差不是 bug，是信仰的裂缝。完美不是没有东西可加，而是没有东西可减。
A 1px misalignment is not a bug, it's a crack in faith. Perfection is achieved when nothing remains to remove.
03
留白即呼吸
Space Is Breath
密集是焦虑的，松弛是自信的。4px 基数体系确保每一次呼吸都节奏一致。
Density is anxiety. Space is confidence. Our 4px baseline ensures every breath follows the same rhythm.
04
克制即力量
Restraint Is Power
全界面只有两种有彩色。90% 中性色 + 10% 强调色 = 100% 清晰。
Two accent colors across the entire surface. 90% neutral + 10% accent = 100% clarity.
风格锚定：Linear 的暗色克制 + Stripe 的排版精度 + Bloomberg Terminal 的金融基因 Anchored by Linear's dark restraint, Stripe's typographic precision, Bloomberg Terminal's financial DNA.

Go/Rust 原生
Go/Rust Native

零 npm 依赖的色值引擎Zero npm dependency color engine

Go 后端开发者不应该为了用一套色值而引入 Node 生态。直接 `go get`，在 main.go 里 import，不需要 package.json。

Go developers shouldn't import Node's ecosystem just to use a set of colors. Just `go get` and import in your main.go. No package.json needed.

Go — 色值常量 + LerpColor + CSS 生成

// go get github.com/libra/go-tokens
import "github.com/libra/go-tokens"

// 涨跌色常量
upColor := tokens.ColorUp // "#ef5350"
downColor := tokens.ColorDown // "#26a69a"

// 颜色插值（gamma-aware）
mid := tokens.LerpColor("#0c0c0e", "#ffffff", 0.5)

// 生成完整 tokens.css（78 个 CSS 变量）
css := tokens.GenerateCSS()
os.WriteFile("tokens.css", []byte(css), 0644)

CLI — 单二进制，零依赖

# go install
go install github.com/libra/go-cli/cmd/libra@latest
libra init # → tokens.css (4289 bytes, 78 CSS vars)

Getting Started

# 在入口文件引入# Import in your entry

# 主题引擎（浏览器环境）# Theme engine (browser)
